## Integrity checks (internal)

This repository includes developer-only integrity checks and protection utilities that run during development and/or build-time. These tools validate that attribution and other repository metadata have not been removed or tampered with.

Important:
- The implementation details of these integrity checks are intentionally not documented here. They are for maintainers and CI only.
- Public documentation will not include step-by-step instructions or internal enforcement rules. Consider these checks as an internal safeguard and contact the repository owner for more information.
