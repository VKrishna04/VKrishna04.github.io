# src/components — overview

This file lists the primary React components and a one-line purpose for each. Use this as a quick reference for contributors.

- `AnimatedBackground.jsx` — Full-screen animated backgrounds; configurable via `settings.json`.
- `FaviconManager.jsx` — Handles dynamic favicon generation/injection.
- `FeaturedProjects.jsx` — Displays the featured projects list on Home/About pages.
- `Footer.jsx` — Site footer content and links.
- `GitHubRepoCard.jsx` — Card for showing GitHub repository info and stats.
- `Navbar.jsx` — Top navigation with mobile menu and unified icons.
- `ParticleBackground.jsx` — Decorative particle background used as a fallback.
- `PrivacyNotice.jsx` — Cookie/privacy notice component.
- `ProjectCard.jsx` — Individual project card used across projects pages.
- `ScrollToTop.jsx` — Small utility component to scroll to top with an icon.
- `TechnicalExperience.jsx` — Displays skills & experience clusters with UnifiedIcon.
- `Navbar/Navbar.jsx` — (internal variant) Duplicate/alternate navbar implementation.

If you want, I can add direct links to each component file and include small code snippets for their public props and example usage.
